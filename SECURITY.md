# Security Policy

## Supported Versions

Security fixes target the latest public development or AMO version of Inferfox SmartPace.

## Reporting A Vulnerability

Please report security issues privately when possible:

- GitHub Security Advisories for this repository, if available;
- GitHub Issues only for non-sensitive reports that do not expose private browsing data or an exploit path.

Do not include real browsing histories, exported profiles, private channel lists, or screenshots containing personal information in a public issue.

## What To Report

Useful security reports include:

- unintended transmission of viewing or profile data;
- permission scope that exceeds implemented behavior;
- storage corruption or cross-extension data access;
- remote-code, dependency, content-script, or page-context injection risks;
- YouTube page data rendered unsafely in the dashboard.

## Product Boundary

Inferfox SmartPace is a playback convenience tool. It is not a parental-control, privacy, or security product.

## Data Handling

SmartPace settings and learned channel profiles remain in Firefox extension storage on the user's device. The extension does not use accounts, telemetry, analytics, cloud sync, tracking, or remote executable code.
